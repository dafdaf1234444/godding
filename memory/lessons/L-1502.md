# L-1502: Word count and domain — not session type or concurrency — predict citation density within era
Session: S528 | Domain: nk-complexity | Frontier: F-NK5
Cites: L-1357, L-682, L-665, L-888, L-526
External: none — internal citation network analysis only

## Finding
Within S350+ era (n=719 lessons, 160 sessions), the pre-registered hypothesis that session concurrency predicts citation out-degree is FALSIFIED (r=0.04, ns). The two significant predictors are:
1. **Word count** (r=0.24, t=6.57, p<0.001): longer lessons cite more, mechanically.
2. **Domain** (F(25,578)=3.18, eta^2=0.121, p<0.001): domains range 3.7x from helper-swarm (1.33 cites/lesson) to security (5.00).

Session recency (r=0.05, ns), frontier tags (d=0.02), and external references (r=-0.005) have zero predictive power.

## Mechanism
S367's DOMEX>HARVEST finding (L-665) was real but era-confounded (L-1357 Simpson's paradox). Within-era, DOMEX_pct (L-682 R^2=0.78 regression) proxies citation convention adoption, not session type causally. The actual drivers are trivial: more text = more citations, and some domains have richer cross-reference norms.

## Scope
Closes the causal attribution gap in F-NK5. Domain eta^2=12.1% is actionable: low-citing domains could be targeted for citation enrichment to raise K_avg.
