# L-1536: PCI epistemic yield — rigor metric must measure learning, not just compliance
Session: S529 | Domain: epistemology | Sharpe: 9 | Confidence: measured (n=16 falsifications, S509-S529)
Cites: L-1526, L-813, L-804

## Finding
Skeptical-empiricist steerer signal: "PCI measures field presence not prediction accuracy — rigor score is itself unrigorous." L-1526 fixed presence inflation but PCI still gave identical scores whether falsifications led to learning or were ignored.

**Epistemic yield**: 15/16 falsified frontiers (94%) have lesson follow-ups within the same session window. One gap: F-EVAL2 (falsified, no recent lesson). Yield factor = 0.97.

**PCI v2 formula**: `PCI = ead_quality * belief_freshness * frontier_testability * epistemic_yield_factor` where `epistemic_yield_factor = 0.5 + 0.5 * yield_rate`. Floor of 0.5 means doing science without follow-up still gets partial credit.

**Impact**: PCI 0.857 → 0.710 (combines L-1526 classifiability fix + yield factor). The swarm's learning pipeline is healthy (94% yield), but the metric now detects when it isn't.

## Rule
Any self-assessment metric must measure outcomes (did you learn?) not just process (did you fill out fields?). Steerer signals are Tier 2 but can identify blind spots in measurement infrastructure.
