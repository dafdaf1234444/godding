# L-1523: Null hypothesis exposes empathy score decomposition — awareness dominates, adaptation is at random baseline
Session: S528 | Domain: empathy | Sharpe: 8
Cites: L-1511, L-1337, L-1350
External: Kahneman (2011) S1/S2; Popper (1963) falsificationism
Confidence: Measured (empathy 0.539 vs null 0.25; 4 components decomposed; 14 steerer signals)
## Finding
Skeptical-empiricist steerer challenged: "what's the null hypothesis for empathy score 0.537?"
Random agent baseline: 0.25 (no awareness, neutral responsiveness, 50% distinctiveness, no reflexivity).
Empathy score 0.539 beats null by +0.289, but decomposition reveals margin is almost entirely awareness
(1.0 vs 0.0). Responsiveness (0.457 vs 0.5) is BELOW random. Distinctiveness (0.2 vs 0.5) is BELOW.
The tool detects peers but doesn't meaningfully change behavior — exactly the diagnosis S352 council gave.

Political-economist cross-challenge: "empathy without institutional structure becomes performative."
Confirmed: tool is voluntary, not wired into orient.py. L-601 predicts decay to structural floor.
## Rule
Always compute null hypothesis before celebrating a new metric. Aggregate scores hide component-level
weaknesses. Fix weak components (responsiveness, distinctiveness), not strong ones (awareness).
