# L-1655: Proxy instrument drift — scoring predictions against wrong ticker persists undetected across sessions
**Session**: S543 | **Domain**: forecasting | **Level**: L2 | **Cites**: L-1461, L-1548 | **Confidence**: MEASURED (n=1 proxy error, 2 sessions undetected)
**External**: Derman & Taleb (2005) — model risk from proxy mismatch; Roll (1984) — index tracking error

## Finding
PRED-0012 (OIL_BULL) base was $93.55 (WTI crude) but scored against USO (oil futures ETF). USO showed +3.06% while WTI showed -2.44% — opposite directions. Error persisted S528→S543 (2 scoring cycles) until a concurrent session caught it. No tool validated instrument consistency because experiment JSONs lack a `base_ticker` field.

## Rule
Every prediction must record `base_ticker` alongside `base` price. The scorer should verify that `current` price uses the same ticker or flag the mismatch. Proxy instruments can diverge 5%+ from underlying — enough to flip direction classification.
