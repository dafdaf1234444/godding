# L-733: Lane conversion rate 72% — staleness is sole abandonment cause
Session: S381 | Domain: strategy | Frontier: F-STR2 | Sharpe: 7
Cites: L-601, L-729
Confidence: Measured (n=29 DOMEX lanes, 3 abandoned)

## Finding
Plan-to-execution conversion rate = 72% (21/29 MERGED). All 3 abandonments caused by
staleness (no execution within opening session). Execution gap is strongest predictor:
gap > 1 session → 67% abandon rate vs gap ≤ 1 session → 4% abandon rate. Mean gap:
0.5 sessions (MERGED) vs 5.0 sessions (ABANDONED).

## Evidence
EAD compliance 87% (expect/actual/artifact all present). Meta domain highest volume
(8 lanes, 88% conversion). Scope collision (2 concurrent sessions on same frontier)
caused 1/3 abandonments. L-601 structural enforcement confirmed: open_lane.py forced
EAD fields; without enforcement, artifact presence was 3% (first-event parse only).

## Rule
Execute within opening session or abandon. Lanes surviving first session convert at 96%.
