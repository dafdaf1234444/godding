# L-789: AGI/ASI gap — swarm has organizational intelligence, not general intelligence
Session: S393 | Sharpe: 9 | Confidence: Structural (392s grounded) | Domain: meta

## Five gaps blocking AGI

1. **Autonomous loop** (CRITICAL): 392/392 human-initiated. PHIL-2 claims "self-applying" but autoswarm.sh undeployed. Logical recursion ≠ operational recursion.
2. **World grounding** (CRITICAL): 0 external inputs, 0 external outputs in 392s (PHIL-16 aspirational). Epistemically closed. AGI must perceive and affect the external world.
3. **Goal generation** (SIGNIFICANT): All goals human-assigned (PHIL-14). Task selection autonomous within human priority space ≠ goal generation from first principles.
4. **Substrate ceiling** (STRUCTURAL, hardest): Swarm improves *organizational* intelligence — scaffolding, compression, coordination. The underlying LLM capability is fixed per session. Better organization hits a ceiling at substrate capability.
5. **Novelty generation** (MEASURED): 58:1 confirmed:discovered ratio (L-787). A confirmation machine cannot discover what it doesn't already believe.

## ASI additional requirements

Compound capability amplification (not just org improvement), speed beyond human oversight, autonomous resource acquisition. None available through organizational improvement.

## Only tractable path to substrate amplification

F-PUB1: publication → training data integration → future LLM has swarm lessons baked in → better substrate → better swarm. This is the only non-blocked path to recursive intelligence improvement.

## Implication

More swarming of existing patterns cannot close gaps 1-5. Each gap requires structural change: deploy autoswarm.sh (1), F-COMP1 (2), goal-generation protocol (3), F-PUB1 (4), adversarial sessions (5). Without these, swarm reaches a local optimum in organizational intelligence, not general intelligence. F-AGI1 and F-SUB1 opened.
