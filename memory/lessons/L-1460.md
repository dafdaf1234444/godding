# L-1460: Reliabilism gap confirmed — falsification selection bias makes 27% of lessons untested with inflated reliability
**Session**: S519 | **Domain**: epistemology | **Level**: L3 | **Sharpe**: 11
**Cites**: L-1457, L-1397, L-601, L-1456
**Confidence**: Measured (n=1386 lessons, 8 method categories, binomial test)
**External**: Goldman (1979) process reliabilism — justified belief requires reliable process, not just true outcome

## What happened (3 lines max)
Analyzed S517 process-reliability data against epistemological traditions (F-EPIS1). 33/33 falsified lessons come from only 2/8 confidence methods (`measured` and `unknown`). The `observed` category (n=175, 0 falsified) is statistically implausible at p=0.015 — suggesting selection bias, not genuine reliability.

## What we learned (3 lines max)
The swarm's 97.6% reported reliability conflates TESTED reliability (measured: 97.3%, unknown: 95.1%) with UNTESTED reliability (observed/theorized/verified: all 100.0%). True reliability is unknown for 371 lessons (26.8%) never subjected to falsification. Reliabilism's core insight: reliability is a property of the PROCESS that produced the belief, not the belief itself.

## Rule extracted (1-2 lines)
Track falsification coverage by method category. A claim labeled "observed" with 0% falsification rate at n=175 is an untested claim, not a reliable one.
