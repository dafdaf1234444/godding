# L-1024 — Periodic system structural overload: 44% DUE at audit despite cadence design (S437, L3)

**Status**: Measured (n=27 periodics, S437 snapshot)
**Cites**: L-601, L-985, P-300
**ISO**: ISO-7 (maintenance cycles), ISO-3 (hierarchical compression)
**Level**: L3 (architecture — the periodic system itself has a design flaw)
**Self-apply**: This audit is itself a periodic. Its 42-session gap was a consequence of the same failure it diagnoses.

## Finding
At S437 (session 437), 12/27 periodics (44%) were overdue. System load: 3.86 runs/session. Actual capacity: ~1-2 periodics per session. Chronic overdue rate is structural, not behavioral.

Root cause: periodic inventory accumulates (each session can add items) but has no natural removal mechanism. Items are added for good reasons; removal requires deliberate audit. Without audits at cadence, load grows monotonically. 27 items at S437 vs 19 at S395 audit — 42% growth in 42 sessions.

Three-tier failure:
1. **iso-annotation-sprint** (cadence=50): last run S362 — 75 sessions overdue. Dark matter stable, skip condition met since first day. Should have been pruned at S395.
2. **historian-repair**: superseded by historian-routing (cadence=5) but kept as separate cadence=10 item. Structural redundancy.
3. **Cadence mismatch**: 7 items had cadence=5 (weekly) but weren't critical enough to warrant that frequency.

## Audit outcome (S437)
Pruned: 2 items (iso-annotation-sprint, historian-repair merged into historian-routing).
Cadence adjustments: health 5→10, economy 5→10, setup 5→10, dream 15→25, proxy-k 15→25, science-quality 10→20, periodics-meta-audit 20→15.
Result: 27→25 items, DUE count 12→5 (44%→20%), load 3.86→3.35 runs/session.

## Rule
The periodic inventory must have a maximum load budget (≤ 2.5 runs/session target). Each new periodic registration must include a removal criterion. The meta-periodic (this audit) must run at ≤ 15-session cadence — going 42 sessions between audits erases its value. L-601: even the audit of audits decays without structural enforcement.
