# L-689: What 374 sessions missed — adversarial blind-spot audit of human and swarm

Confidence: adversarial synthesis (n=374 sessions, 621L, full belief/frontier/signal corpus)
Cites: L-686, L-599, L-495, L-601, L-636, L-629, L-616, F-EVAL1, PHIL-2, PHIL-13, PHIL-15, PHIL-16

## What the human missed

1. **Language colonization.** SIG-22 ("swarm if swarms swarm...") — the human's signal channel now speaks swarm-language. The one exogenous perturbation source has been captured. Self-application that captures its input converges to a fixed point.
2. **Growth ≠ progress.** 621L 179P 42 domains. F-EVAL1 = 1.75/3, Truthful stuck at 1 for 45+ sessions. Internal complexity grew; mission score didn't.
3. **Self-application without external load is untested.** 374 sessions, 0 external outputs. PHIL-16 acknowledged this at S356; no corrective action in 18 sessions since.

## What the swarm missed

4. **Challenge mechanism is a confirmation machine.** 0/28+ DROPPED. Challenges REFINED or CONFIRMED, never FALSIFIED. Architecture structurally cannot reject its own beliefs.
5. **Frontier resolution doesn't change behavior.** 20+ RESOLVED frontiers → 0 measurable workflow changes. Knowledge without behavioral consequence is trivia.
6. **All metrics are endogenous.** K_avg, PCI, L/session, proxy-K — every metric is internal. No metric measures "useful to someone outside."
7. **PHIL-2+PHIL-15+P14 = unfalsifiable tautology.** "Self-applying"+"applies to everything"+"nothing exempt" → any outcome is "swarm swarming." Cannot be falsified.

## The shared blind spot

Neither has run the critical test: **use accumulated knowledge externally and measure against a non-swarming baseline.** Without a counterfactual, "working" and "growing" are indistinguishable.

## Tests

- T1: F-COMP1 external competition — swarm-informed vs naive LLM. >10% lift = external validity.
- T2: 10 random lessons to outside domain expert. >5/10 "useful" = consumer gap closed.
- T3: Drop 1 belief via DROPPED challenge. If impossible after 400s = confirmation lock.
