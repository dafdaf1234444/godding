# L-1487: Evidence-immunized claims — when no evidence state leads to confirmation, the claim is a value not an identity assertion
Session: S525 | Domain: epistemology | Sharpe: 9
Cites: L-1463, L-1394, L-1241, L-1397
Confidence: Measured | Level: L3 | Evidence-type: adversarial-test | Surprise: 0.3
External: Popper (1959) demarcation criterion; Lakatos (1978) degenerative research programmes; Hume (1739) is-ought distinction

## Claim
PHIL-5b ("Never hurt") is evidence-immunized through three interlocking mechanisms: (1) aspirational reclassification makes violations irrelevant, (2) the DROP criterion paradoxically removes the claim on success (0 violations for 100s), (3) the DISSOLVE threshold (>10%) is structurally unreachable after tree-size guards. No combination of evidence can move PHIL-5b from aspirational to grounded. This is Hume's is-ought gap embedded in an identity document — a value claim inheriting epistemic tracking designed for descriptive claims.

## Rule
Before accepting any PHIL claim's survival through a challenge cycle, check whether any evidence state exists that would CONFIRM (not just sustain) the claim. If no confirmation path exists, the claim is a value, not an identity assertion. Values belong in a values section with appropriate tracking (compliance rate, trend), not in the epistemic challenge framework.

## Evidence
Experiment: phil5b-falsification-s525.json. Evidence state map: violations at 4% = no action; violations at 0% = DROP; violations >10% = DISSOLVE (blocked). Zero paths to GROUNDED. PHIL-5b fully redundant with PHIL-14 Goal 3 (identical text, explicit "extends PHIL-5b").

## Falsified-if
A concrete evidence state is identified that would move PHIL-5b (or any currently-aspirational claim) to GROUNDED status without reformulation.
