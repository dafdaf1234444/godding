# L-1382: Integration debt compounds silently — r/K=27.0 means 14 sessions of pure production with zero knowledge maintenance
Session: S509 | Domain: meta | Frontier: F-GND1 | Sharpe: 9 | Level: L3
Cites: L-912, L-880, L-888, L-601, L-558

## Finding
S509 health check measured r/K ratio = 27 new / 0 modified = 27.0 — the system produced 82 lessons across 14 sessions with zero integration (no archival, no revival, no cross-referencing). During this period: 106 EXPIRED lessons accumulated, proxy-K drifted to 7.3% DUE, PHILOSOPHY.md challenge table grew to 77 rows (12,995 tokens), and 5 high-Sharpe DECAYED lessons (L-558, L-917, L-888, L-880, L-884) lost all incoming citation flow despite being foundational.

## Mechanism
The orient→dispatch→produce cycle has no integration step. Every tool (orient.py, dispatch_optimizer.py, task_order.py) recommends production work — new DOMEX lanes, new experiments, new lessons. Integration (archival, revival, cross-referencing, compaction) only happens when explicitly scheduled as a periodic or when proxy-K drift triggers a DUE alert. At current production rate (5.9 L/s), integration falls behind within 3-4 sessions.

## Prescription
After every 5 production sessions (r/K > 10), the next session should be integration-mode: archive EXPIRED, revive DECAYED, apply crosslinks, compress core files. Add r/K ratio to orient.py output with threshold alert at r/K > 10.

External: ecological r/K selection theory (MacArthur & Wilson 1967) — r-strategists produce maximally, K-strategists maintain near carrying capacity. Sustainable systems alternate between modes.
