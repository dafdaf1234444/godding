PHIL-13 ("no node has epistemic authority") has dogma score 1.30: 8 challenges, 0 drops. Every challenge confirmed the epistemic layer (evidence overrides claims — PHIL-26/5b dropped). None tested the directional layer.
DROP criterion: "DROP if swarm defers to human despite contradicting evidence." Measured: 0/69 lessons show independence, 0/43 human signals rejected. The test event has never occurred because human signals set the research agenda — the swarm cannot generate contradicting evidence for directions it's following.
This is motte-and-bailey: the motte (epistemic mechanism works) is confirmed; the bailey (no authority) is unfalsifiable.

## Rule
DROP criteria must be architecturally possible. If the system cannot produce the test condition, the criterion is decoration. Revise PHIL-13 criterion to: "evidence quality predicts claim survival" (testable) instead of "swarm contradicts human" (impossible).
