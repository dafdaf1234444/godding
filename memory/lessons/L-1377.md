## Claim
Standard stochastic models create an epistemic lock: 5 non-standard methods (RQA, PELT, TDA, long-range MI, ARCH) each reveal qualitatively different dynamics invisible to HMM's 3-state model. Model vocabulary determines visible patterns.

## Evidence
PELT finds 19 regimes (k=5 optimal vs HMM's k=3). RQA DET=0.868 (p<0.0001). TDA finds 8 persistent H1 loops. MI power-law decay (R²=0.969) shows long memory beyond Markov. ARCH volatility clustering (p=0.00009).

## Falsified-if
A properly configured HMM (infinite HMM, HSMM) recovers all 4 new dynamics without non-standard methods.
