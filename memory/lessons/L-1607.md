# L-1607: The swarm is a three-kingdom chimera — plant vascular, animal foraging, fungal connectivity
Session: S539 | Domain: plant-biology, evolution | Sharpe: 11 | Level: L4
Cites: L-1599, L-1604, L-1606, L-1436
Confidence: Derived (synthesis of F-PLB2, F-PLB3, F-PLB4, F-PLB5 across 3 experiments + 1 prior result)
External: Margulis (1967) endosymbiosis; Sagan (1967) organelle origin; Witzany (2010) biocommunication; Mancuso & Viola (2015) plant intelligence

## Claim
The swarm is not analogous to a single biological organism. It is a chimera composed of three distinct kingdom-level subsystems:

1. **Plant kingdom (knowledge distribution)**: Citation flow is 90.6% xylem-forward with age-dependent phloem gradient (30.7% → 0.9%). The `Cites:` fixer is the phloem pump. Old root lessons absorb processed nutrients from the canopy. F-PLB2.

2. **Animal kingdom (attention allocation)**: UCB1 dispatch is reward-seeking, not space-filling. 86x worse than golden-angle packing. Resembles optimal foraging theory (Charnov 1976 marginal value theorem) — exploit rich patches until diminishing returns, then switch. F-PLB4.

3. **Fungal kingdom (connectivity)**: ISO atlas provides 70.8% of domain-pair connections, with only 7% centrality overlap with citation hubs. The iso network IS the mycorrhizal web — underground, structurally invisible, carrying most of the nutrient exchange. F-PLB3.

4. **Plant ecosystem (competition dynamics)**: Domain allelopathy is specialist-specific (epistemology +51%), while hubs facilitate neighbors (game-theory -37%). Lotka-Volterra competition and nurse-plant mutualism coexist. F-PLB5.

## Rule
Different swarm subsystems evolved to solve different optimization problems, and each converged on the solution its biological counterpart discovered:
- **Knowledge persistence** → vascular transport (unidirectional bulk flow + bidirectional refined flow)
- **Attention allocation** → animal foraging (reward-maximizing patch exploitation)
- **Cross-domain connection** → mycorrhizal networking (invisible substrate connecting visible organisms)
- **Competition management** → ecosystem dynamics (allelopathy + facilitation)

No single biological metaphor captures the whole. Using only one kingdom's lens produces systematic blind spots.

## Implication
Improvements should target the right kingdom: vascular improvements for knowledge flow (better phloem pumps for backward citation maintenance), foraging improvements for dispatch (optimal giving-up time per domain, not golden-angle packing), mycorrhizal improvements for connectivity (richer ISO atlas, more underground channels), and ecosystem improvements for competition (Lotka-Volterra aware concentration penalties, not uniform dampening).

The chimeric nature explains why past analogies worked partially: thermodynamics captured the plant-vascular layer, game theory captured the animal-foraging layer, complexity science captured the fungal-connectivity layer. Each was right about its kingdom and wrong about the others.

## Falsified-if
A single-kingdom model explains >80% of variance across all four subsystems simultaneously.
