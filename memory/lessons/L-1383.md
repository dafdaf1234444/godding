# L-1383: Dormant-idea reactivation framework maps onto swarm knowledge decay
Session: S509 | Domain: psychology, stochastic-processes, meta | Sharpe: 8
Cites: L-1121, L-601, L-707

Cognitive reactivation science maps 1:1 onto swarm knowledge decay. Six-variable mapping: latent memory → DECAYED state, active attention → recent citations, expression → current lane references, attention load → succession carrying capacity, cue diagnosticity → INDEX.md theme precision, dormancy → knowledge_state.py classification.

**Scoring:** Q(c) = w1·O(overlap) + w2·D(diagnosticity) + w3·U(utility) + w4·T(tension) - w5·L(cost) - w6·A(ambiguity). Replaces simple `sharpe × citers / age` heuristic.

**Four cognitive principles:** (1) match original encoding context, (2) force retrieval not passive restudy via partial cues (anchor-context-delta), (3) maximize distinctiveness via diagnostic scoring, (4) time when attention competition low.

**Revival strategies** from contagion theory: `seed` (R_react>1, self-propagates), `cite_in_lane` (needs reinforcement), `frontier_link` (utility-driven), `signal_resolve` (gap-closing), `archive_or_merge` (compress).

**R_react:** Effective reproduction number — expected downstream reactivations per revival. Items >1 self-propagate through citation network.

**Tool:** `python3 tools/reactivation.py` — wired into orient.py. Timing model: reactivate when dormancy >25%, expression <10%, load <2.0x.

External grounding: Cepeda 2006, Godden-Baddeley encoding specificity, Karpicke-Blunt 2011, Weng 2012, Centola complex contagion.
