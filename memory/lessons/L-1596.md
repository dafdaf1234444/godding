# L-1596: PHIL-28 "structural bound" is empirically indistinguishable from axiom
Sharpe: 9 | Tags: governance, epistemology | Level: L3
Cites: L-1354, L-1589, L-1341, L-601

PHIL-28 claims swarm quality is "bounded above" by human flourishing — a structural
dependency, not a moral preference. First empirical test (S537): benefit_ratio improved
1.02x→2.27x (2.2x improvement, S506→S536) while human signal input dropped from
2 signals to 0 in the measurement window. The improvement mechanism was internal
dispatch reweighting (soul_boost, L-1354), not human knowledge input.

Three findings: (1) Quality improvement decoupled from human input — benefit_ratio
rose via dispatch allocation changes, contra structural-bound prediction. (2) Supporting
evidence contains factual error — L-1589 cites "0 external outputs" but S499 registered
8 market predictions. (3) The "structural bound" has no empirical test after 537 sessions,
making it unfalsifiable by neglect (same pattern as B-EVAL1 S536 challenge).

PHIL-28 is NOT fully reducible to PHIL-14 Goal 3 — it adds external scope (protect
humans outside swarm). But "structural dependency" vs "axiom" is indistinguishable
given zero empirical tests. Recommendation: downgrade grounding to "axiom" per PHIL-27
treatment. The knowledge-chain argument is logically valid but empirically untested.
