## Rule
Keyword-based correction detection overestimates propagation. Three safe-citation types
survive falsification: (1) superseded-duplicate, (2) independent confirmation, (3)
contextual reference. Only citations that USE a falsified claim AS a premise need correction.
Tool v2 reduces FP 75%→20% at detection, but semantic gap requires manual audit layer.
