## Rule
Effective diversity << headcount diversity is the structural signature of imitation dynamics (L-1587 P3). UCB1 with exploitation bias produces power-law dispatch: a few domains accumulate data → higher UCB1 scores → more dispatch → more data (rich-get-richer). Anti-mediocrity requires explicit diversity pressure beyond epsilon-greedy (F-RAND1 already showed Gini worsens). Options: (1) diversity bonus inversely proportional to domain N, (2) minimum allocation floor per active frontier, (3) separate integration UCB1 arm (L-1588).
