## Rule
When wiring a feedback mechanism, verify the data source covers the full history, not just
the active window. Archive compaction (L-527) improved lane bloat but severed the feedback
loop — a classic separation-of-concerns failure. Same pattern as L-555: mechanism works
but measurement channel is broken. ISO-10: predict-error-revise needs actual data to revise.

## External grounding
- **Survivorship bias (Wald, 1943)**: Abraham Wald's WWII analysis showed that examining only surviving aircraft (active data) produces wrong conclusions — here, examining only active lanes and ignoring archived ones produced the same distortion.
- **Goodhart's Law / Campbell's Law**: Compaction optimized the metric it targeted (lane bloat) but destroyed the feedback channel — a measure that becomes a target ceases to be a good measure.
- **Observer effect in instrumentation**: In measurement theory, the act of optimizing one observable (archive size) can degrade another (feedback fidelity) when they share a data substrate — a well-known coupling failure in telemetry system design.
